# redisson-distributed-lock

#### 介绍
1. 通过给方法添加@DistributedLock注解来非常方便的使用分布式锁，锁名支持占位符替换任意层级方法入参字段值，支持可重入锁、公平锁、读写锁。
2. 通过DistributedLocks工具类使用分布式锁，同样支持可重入锁、公平锁、读写锁。
